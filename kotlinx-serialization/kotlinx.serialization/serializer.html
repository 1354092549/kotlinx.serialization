(jvm) fun serializer(type: Type): KSerializer<Any>

Constructs a serializer for the given reflective Java type. serializer is intended to be used as an interoperability layer for libraries like GSON and Retrofit, that operate with reflective Java Type and cannot use typeOf.

For application-level serialization, it is recommended to use serializer<T>() instead as it is aware of Kotlin-specific type information, such as nullability, sealed classes and object.

(jvm) fun SerializersModule.serializer(type: Type): KSerializer<Any>

Retrieves serializer for the given reflective Java type using reflective construction and contextual lookup for non-serializable types.

serializer is intended to be used as an interoperability layer for libraries like GSON and Retrofit, that operate with reflective Java Type and cannot use typeOf.

For application-level serialization, it is recommended to use serializer<T>() instead as it is aware of Kotlin-specific type information, such as nullability, sealed classes and object singletons.
