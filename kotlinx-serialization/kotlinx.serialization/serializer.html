(common, jvm, native) fun serializer(type: KType): KSerializer<Any?>(common, native)

Creates a serializer for the given type with support of user-defined generic classes. type argument can be obtained with experimental typeOf method.

Example of usage:

val map = mapOf(1 to listOf(listOf("1")))
val serializer = serializer(typeOf<Map<Int, List<List<String>>>>())
json.encodeToString(serializer, map)

(jvm) fun serializer(type: Type): KSerializer<Any>

Constructs a serializer for the given reflective Java type. serializer is intended to be used as an interoperability layer for libraries like GSON and Retrofit, that operate with reflective Java Type and cannot use typeOf.

For application-level serialization, it is recommended to use serializer<T>() instead as it is aware of Kotlin-specific type information, such as nullability, sealed classes and object.
