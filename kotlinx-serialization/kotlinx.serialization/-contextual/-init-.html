<init>

(common, jvm, native) <init>()

Instructs the plugin to use ContextSerializer on a given property or type. Context serializer is usually used when serializer for type can only be found in runtime. It is also possible to apply ContextSerializer to every property of the given type, using file-level UseContextualSerialization annotation.
