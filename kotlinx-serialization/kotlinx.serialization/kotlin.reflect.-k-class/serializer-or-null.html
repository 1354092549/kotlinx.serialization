(native) fun <T : Any> KClass<T>.serializerOrNull(): KSerializer<T>?

Retrieves a KSerializer for the given KClass or returns null if none is found. The given class must be annotated with Serializable or be one of the built-in types. It is not recommended to use this method for anything, but last-ditch resort, e.g. when all type info is lost, your application has crashed and it is the final attempt to log or send some serializable data.

This API is not guaranteed to work consistent across different platforms or to work in cases that slightly differ from "plain @Serializable class".
