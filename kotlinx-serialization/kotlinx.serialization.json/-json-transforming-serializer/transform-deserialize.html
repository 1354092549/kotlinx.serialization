transformDeserialize

(common, jvm, native) protected open fun transformDeserialize(element: JsonElement): JsonElement

Transformation that happens during deserialize call. Does nothing by default.

During deserialization, a value from JSON is firstly decoded to a JsonElement, user transformation in transformDeserialize is applied, and then resulting JsonElement is deserialized to T with tSerializer.
